# Kontimer

A Desktop Timer App Written in Flutter And Dart. 

## Getting Started
  
### Requriments
To Install you will need to have the latest flutter version
### Commands

#### Windows
flutter build windows

#### Macos
flutter build macos

#### Linux
flutter build linux
