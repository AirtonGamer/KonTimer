# Kontimer

A Desktop Timer App Written in Flutter And Dart. 

## Getting Started
  
### Requriments
- The Latest Flutter Version

- For Linux you need to install gstreamer if not installed

### Commands

- #### Windows
```dart 
flutter build windows
```

- #### Macos
```dart 
flutter build macos
```

- #### Linux
```dart 
flutter build linux
```
